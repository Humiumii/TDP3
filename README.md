# Fruit Classifier — Clasificador de Calidad de Frutas

Clasifica fotos de **manzana, plátano y naranja** en estado **bueno** o **dañado** usando visión por computadora.

## Stack
- Python + PyTorch (MobileNetV3)
- Streamlit (Web app)
- GitHub (versionado)

## Estructura
```
fruit-classifier/
├── src/          # Código fuente (entrenamiento, predicción, utils)
├── docs/         # Documentación del proyecto (rúbrica)
├── data/         # Dataset
├── notebooks/    # Jupyter notebooks
├── models/       # Pesos entrenados
└── requirements.txt
```
